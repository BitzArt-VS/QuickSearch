# GUI Framework — Overview

Blazor-inspired, Cairo-backed. Components declared in C# via `BuildRenderTree`. Lives in `src/UI-Tweaks/Gui/Framework/`.

**Key terms:** Component — unit of UI. RenderFragment — stable `delegate void(IGuiRenderTreeBuilder)` describing a subtree; `GuiRenderFragment<T>` is the parameterised counterpart `delegate void(IGuiRenderTreeBuilder, T)` (flat, not curried like Blazor's — saves an allocation per item per rebuild while keeping the same call-site DX). Builder — receives declarations, owns instances. Reconciler — diffs declarations vs live instances. Renderer — tracks reconcile/arrange/paint invalidation, runs the dirty combined arrange+paint walk, and blits the latest texture each frame.

```
Public/Components/Base/                   GuiNode                  lightweight base: render handle + render fragment + lifecycle + cascade lookups + protected Slot helper + ConfigureSlot + BuildRenderTree hook + Render/RenderOverlay (no-op). No layout — use for layout-transparent decorators.
                                          GuiComponent : GuiNode   default base for layout-participating components; adds LayoutParameters + virtual Measure through IGuiMeasurable; default Measure recursively measures mounted child slots through the protected Slot helper (backed by IGuiRenderHandle.Slot) using stack-layout rules
Public/Components/Layout/                 GuiContainer : GuiComponent  general-purpose layout/surface ("the div"): hosts Content children; optional Background fill (default transparent → no draw); subclasses override DrawBackground (before children) + DrawOverlay (after children); base Render/RenderOverlay are sealed; defaults to FitContent
                                          GuiInset : GuiComponent      vanilla inset visual: optional dark brightness overlay + multi-pass embossed rounded-rectangle border; Depth controls ring passes, Brightness the fill overlay, Radius corner curvature; Content children render on top
                                          GuiShadow : GuiComponent     soft drop shadow: stack of offset, decreasing-alpha rounded rectangles toward bottom-right; Steps + Offset + Spread configurable; also exposes a static Draw helper for call sites that want the shadow without a tree slot (e.g. GuiSlider handle)
Public/Components/Display/                GuiLabel                     sealed leaf: renders a single text line via GuiFontStyle; Measure() returns intrinsic text size; default font = GuiFontStyle.Default
                                          GuiRectangle : GuiComponent  leaf: fills its bounds with a solid Color (GuiColor, default transparent → no-op draw); both axes default to FitContent
                                          GuiSeparator : GuiRectangle  sealed pre-styled horizontal rule: 1 px tall, widthMode = Fill, Color = GuiVanillaStyle.DialogTitleBarBgColor; defaults registered via ConfigureSlot; use AddSeparator(key, margin?) extension
                                          GuiBreadcrumbs<T> : GuiComponent  horizontal breadcrumb trail: renders PreviousItems via ItemTemplate (GuiRenderFragment<T>) with " > " label separators, then CurrentItem via CurrentTemplate. With no PreviousItems collapses to a single-slot heading (CurrentTemplate only, with section-spacing bottom margin). Click and hover subscriptions belong inside ItemTemplate — chain OnMouseEnter/OnMouseLeave/OnMouseClick on any declared slot; the component wraps each item in a scoping container for key isolation only. RenderingItemIndex (protected) is set before each ItemTemplate call so subclass templates closing over `this` know which item index is currently rendering. Non-generic GuiBreadcrumbs : GuiBreadcrumbs<string> provides predefined ItemTemplate (HyperlinkColor label + HyperlinkHoverColor on hover + linkselect cursor + OnMouseClick → OnItemClicked action; hover tracking via a wrapper container owning all three events so the label itself carries no interactive region) and CurrentTemplate (LargeBold label).
Public/Components/Controls/               GuiButton                    sealed clickable: vanilla-style Normal button visuals; registers own-slot OnMouseDown/Up/Click; exposes only OnClick (GuiCallback)
                                          GuiMouseTarget               sealed layout-transparent GuiNode used as an explicit mouse-event target around visible content; wraps Content and derives hit bounds from the wrapped child extent
Public/Components/Dialog/                 GuiDialog                    default base implementation of IGuiDialog and IGuiResizable; explicitly implements runtime attachment, tracks native dialog focus through a root-slot focus callback, and exposes RequestClose/RequestFocus helpers plus resize policy; concrete dialogs subscribe to root-slot key/focus events through ConfigureSlot when needed; OffsetX/OffsetY (logical px from screen-centred default) control dialog position — mutated by drag; Move(deltaX, deltaY) adds a delta (pass as GuiDialogTitleBar.OnDrag to make the title bar drag the dialog); IsResizable enables bottom/right edge resize by default with Min/Max Width/Height bounds, while internal GuiResizeController owns gesture mechanics
                                          GuiDialogBackground : GuiContainer  vanilla shaded dialog body: solid FillColor fill + tiled Texture overlay + outer border stroke; configurable FillColor, Texture, TextureAlpha, TextureScale, BorderColor, BorderAlpha, BorderWidth + JoinedTopEdge flag (suppresses top stroke to butt against a title bar above)
                                          GuiDialogTitleBar : GuiContainer    vanilla shaded title bar: lighter solid fill + open three-sided border (left + top + right); Title text drawn directly by DrawBackground (vertical centering not yet in layout pass); JoinedBottomEdge on GuiDialogBackground closes the bottom; OnDrag (Action<double,double>?) — when set the bar acts as a drag handle: pass `this.Move` from a GuiDialog subclass to wire title-bar drag to the dialog's position (null by default — drag disabled)
                                          GuiDialogCloseIcon : GuiComponent   vanilla "X" close icon: cross + drop shadow drawn into the Cairo surface every frame; hover changes cross colour instead of vanilla's pre-baked texture overlay; exposes CrossSize + CrossLineWidth; bounds sized to fit the full visual (cross + shadow) exactly
Public/Components/Tooltip/                GuiTooltip : GuiNode         sealed layout-transparent wrapper: implements only IGuiNode (not IGuiComponent), so the layout pass inlines its Content children at the parent's flow and the wrapper itself occupies no space. In Render registers itself with the cascaded TooltipHost so the floating layer can draw TooltipContent over its bounds (= union of inner children along parent's flow axis)
                                          GuiTooltipBackground : GuiContainer  vanilla tooltip chrome: DialogStrongBgColor fill + DialogBorderColor stroke at width 3, radius 1, 5px padding (matches GuiElementHoverText.DefaultBackground)
Public/Components/Inputs/                 GuiInputBase : GuiComponent  shared input boilerplate: cascade-resolves FocusManager + registers own-slot click target callbacks + tracks IsHovered/IsPressed/IsFocused/LastBounds; subclasses override OnInputClick / OnInputMouseDown / OnInputMouseUp / OnInputMouseMove plus Render for chrome
                                          GuiTextInput / GuiCheckbox / GuiSlider  sealed inputs over GuiInputBase
                                          GuiDropdown<T> : GuiInputBase  generic picker: header (selected-template + chevron) + a popup (GuiDropdownPopup, internal subclass of GuiTooltipBackground) listing rows via ItemTemplate (GuiRenderFragment<T>); falls back to item.ToString() label when no template; SelectedIndex is canonical state, Up/Down arrow keys navigate, Space/Enter toggle, Escape closes; auto-closes on focus loss; rebuild-on-hover for row highlight; extension hooks BuildHeader / BuildPopup / GetItemCount / GetItemAt for autocomplete-style overrides. Popup is hoisted onto the dialog's OverlayHost (see Internal/Renderers/) so it paints on its own Cairo surface, can hang off the dialog's edges without being clipped, and its rows always win the topmost-wins reverse hit-test. Non-generic GuiDropdown : GuiDropdown<string> exists for convenience.
Public/Interfaces/                        IGuiNode                     public minimal node contract: RenderFragment, Attach, lifecycle, Render/RenderOverlay. No layout — slots whose instance only implements this are layout-transparent (child builder inlined at parent's level, wrapper bounds = union of inner children)
                                          IGuiMeasurable               public small capability contract containing only Measure(GuiLayoutSize available)
                                          IGuiComponent : IGuiNode, IGuiMeasurable  public layout-participating contract: adds LayoutParameters and inherits Measure
                                          IGuiResizable : IGuiMeasurable  public resize capability contract: SupportedResizeEdges + Resize(GuiComponentBounds), intentionally not tied to IGuiComponent
                                          IGuiComponentSlot            public read-only mounted-tree view exposed through IGuiRenderHandle.Slot; exposes Node + child slots without renderer internals
                                          IGuiDialog : IGuiComponent  public renderer-facing dialog contract: OffsetX/OffsetY plus optional `AttachDialogRuntime`; resource cleanup is opt-in through normal IDisposable implementation on the concrete dialog/component
                                          IGuiDialogRuntime          public runtime bridge passed to dialogs by the renderer; exposes FocusedNode plus RequestClose/RequestFocus/SetFocusedNode/SetMouseOverCursor for custom IGuiDialog implementations
                                          IGuiRenderTreeBuilder        builder contract; exposes AddComponent<T> (internal, where T : IGuiNode) + PushCascadeScope<T> (internal, drives cascade scopes)
                                          IGuiSlotBuilder              shared slot configuration contract; used by external component builders and ConfigureSlot; backs ConfigureLayout plus OnMouse*/OnKey* extensions
                                          IGuiComponentBuilder<T>      fluent external slot handle (T : IGuiNode); extends IGuiRenderTreeBuilder + IGuiSlotBuilder
                                          IGuiRenderHandle             exposes Slot, RequestReconcile(fragment), RequestArrange(), RequestPaint(), compatibility RequestRender(), TryGetCascadingValue<T>([name,] out T) — cascade lookups
Public/Enums/                             GuiDirection, GuiComponentPositioning, GuiHorizontalAlignment, GuiResizeEdge, GuiScrollDirection, GuiSide, GuiSizeMode, GuiTextInputMode, GuiVerticalAlignment
Public/Models/                            GuiComponentLayoutParameters, GuiMouseEventArgs (record struct: Position + AbsolutePosition — both GuiPoint in logical px; Button; body property WheelDelta — non-zero only for wheel events, zero for all click/move events), GuiKeyEventArgs (readonly struct — wraps vanilla KeyEvent; Handled flag propagates back to inner KeyEvent)
Public/Common/                            GuiComponentBounds, GuiThickness, GuiColor, GuiSize (Auto/fixed/fractional layout constraint; `int`/`double` convert to fixed, strings like `"50%"` convert to fractional, `Fraction(...)` resolves during arrange), GuiLayoutSize (Width/Height pair used as both measurement input and output), GuiComponentLayout (public measurement helper: MeasureContent over child slots + ResolveAllocatedSize for fixed/fraction/fill/fit sizing), GuiPoint (readonly record struct — X/Y in logical px; used as Position/AbsolutePosition in GuiMouseEventArgs), GuiFontStyle, GuiVanillaStyle (framework mirror of vanilla GuiStyle — all values pulled from GuiStyle at type-init so they never drift; covers: backgrounds: DialogLightBgColor/DefaultBgColor/StrongBgColor/BorderColor/DialogShadedStrokeColor/DialogTitleBarBgColor/DialogTitleBarHighlightColor; foregrounds: DialogDefaultTextColor/DarkBrownColor/DialogHighlightColor/DialogAlternateBgColor; buttons: ButtonBackColor/ButtonTextColor/ActiveButtonTextColor; status: SuccessTextColor/ErrorTextColor/WarningTextColor/LinkTextColor/HyperlinkColor/HyperlinkHoverColor; textures: DirtTexture/NoisyMetalTexture/WoodTexture/StoneTexture/WaterTexture/PaperTexture; dimensions: TitleBarHeight/DialogBgRadius/ElementBgRadius/ElementToDialogPadding/DialogToScreenPadding/HalfPadding; framework accent tokens: SwitchMarkColor (steel-blue #7AAEC2 — checkbox tick, slider mark), SliderFillColor (same hue at ~40% alpha for wide track fills)), TooltipHost (per-dialog tooltip controller, published as a root cascading value; wraps the tooltip FloatingLayerRenderer), OverlayHost (per-dialog overlay controller for popup-style content, published as a root cascading value; wraps the overlay FloatingLayerRenderer; Show(token, bounds, fragment) / Hide(token)), FocusManager (per-surface focus controller, published as a root cascading value; RequestFocus(node) / Blur(); FocusedNode tracks focused IGuiNode by reference; framework auto-blurs on click-outside and surface close), GuiCursorHost (per-surface cursor controller, published as a root cascading value; SetHoverCursor(code) / null to clear; GuiInputRouter reads HoverCursor after each mouse-move and forwards to the adapter; override cursors such as resize take priority)
Public/Services/                          GuiDialogHost (static singleton facade; Initialize(clientApi) creates the default host instance; Toggle/Open/Close/IsOpen/TryGet delegate to Instance; TDialog is any class IGuiDialog with a public parameterless constructor), GuiDialogHostInstance (host-owned dialog renderer registry; public constructor creates isolated hosts; Open creates a typed DialogRenderer<TDialog> or in-place reconciles an already-open configured dialog; Close removes the renderer; mounted dialog/tree slots are disposed only when their concrete nodes implement IDisposable)
Public/Extensions/                        RenderTreeBuilderExtensions  Add* slot declarations; Add<T> accepts any IGuiNode (layout args only apply to IGuiComponent slots) + AddButton/OnClick + AddCascadingValue<T> (void scope wrapper — no layout params, no key) + AddTooltip (wraps content; tooltip shown via cascaded TooltipHost; returns IGuiComponentBuilder<GuiTooltip> for chaining)
                                          ComponentBuilderExtensions   Configure (T : IGuiNode) / ConfigureLayout (T : IGuiComponent)
                                          MouseEventBuilderExtensions  OnMouseDown / OnMouseUp / OnMouseClick / OnMouseMove / OnMouseEnter / OnMouseLeave (T : IGuiNode; Action<T> + Func<T,Task> overloads)
                                          KeyEventBuilderExtensions    OnKeyDown / OnKeyUp / OnKeyPress (T : IGuiNode; Action<GuiKeyEventArgs> + Func<GuiKeyEventArgs,Task> overloads); keyboard events fire on the root dialog slot first, then on the currently focused slot (not spatially routed); handlers may set GuiKeyEventArgs.Handled to suppress framework fallbacks
                                          FocusEventBuilderExtensions  OnFocusChanged (T : IGuiNode; Action<bool> + Func<bool,Task> overloads); focus events fire when the slot's node becomes or stops being the focused node; on the root dialog slot they track native dialog/surface focus
                                          CairoContextExtensions       low-level Cairo drawing primitives: RoundRect (path append), fill/stroke helpers, SetPatternSource (tiled texture with counter-scaled CTM); all coordinates in logical pixels
Public/Delegates/                         GuiRenderFragment, GuiRenderFragment<T> (flat (builder, item) shape — see Key terms), GuiCallback, GuiCallback<T>
Internal/Services/                        GuiRenderTreeBuilder         reconciler + builder; PushCascadeScope<T> does a stack push/pop during blueprint phase; CascadeChain snapshotted per-frame; owns keyed component slots declared through normal fragments; emits InteractiveRegions during the combined arrange+paint walk; layout pass branches on `slot.Instance is IGuiComponent` — pure IGuiNode slots are inlined at the parent's level
Internal/Services/                       GuiInputRouter               generic per-surface input router; receives native mouse/key/focus/Escape events from GuiElementAdapter, owns interactive/keyboard regions, mouse capture, hover transitions, root-slot key dispatch, focused-node key dispatch, cursor priority, surface/overlay containment handling, host-focus/cursor delegates, and the default Escape fallback
                                          GuiResizeController          internal resize gesture controller owned by GuiInputRouter; owns edge hit-testing, cursor selection, capture-like gesture state, root-screen bounds conversion, and dispatches suggested GuiComponentBounds to IGuiResizable
Internal/Renderers/                       GuiSurfaceRenderer           abstract base for any Cairo-backed component tree renderer: implements IDisposable; directly owns Cairo surface (ImageSurface + Context + LoadedTexture lifecycle) + GuiRenderTreeBuilder, plus scale tracking and dirty flag; virtual members Schedule/Cancel/Add*Region/ContainsScreenPoint — all no-op by default; protected methods EnsureSurfaceSize / DrawSurfaceContents / BlitAt; internal SetCascadeChain
                                          DialogRenderer / DialogRenderer<TDialog>  frame-loop entry point split into a type-erased base and typed dialog owner; extends GuiSurfaceRenderer; TDialog is constrained to class IGuiDialog rather than GuiDialog; typed renderer declares the dialog as a normal keyed slot, validates fixed root dimensions after each dialog reconciliation, attaches GuiDialogRuntime through IGuiDialog.AttachDialogRuntime, renders the dialog through the normal Builder.Render/Builder.Paint path, OnRenderFrame drives the real frame loop, Schedule/Cancel → ScopedRebuildQueue, Add*Region → GuiInputRouter, and directly handles dialog-local ↔ screen coordinate conversion; owns GuiInputRouter + two FloatingLayerRenderer sub-layers iterated directly for the OnFrameStart / RunWalk / Render lifecycle
                                          FloatingLayerRenderer        internal non-sealed driven sub-layer; extends GuiSurfaceRenderer with active-fragment storage + Show/Hide/IsActive for external control; wraps active fragments in a normal root GuiContainer slot before measuring/rendering; per-frame driven lifecycle (OnFrameStart / RunWalk / Render → Update + Blit); placement configured via FloatingLayerPlacement (Anchor, size constraints, InputHost for region forwarding, AutoClearWhenNotRefreshed, RewalkOnDialogWalk); the tooltip layer and overlay layer inside DialogRenderer are both FloatingLayerRenderer instances differentiated solely by their FloatingLayerPlacement
Internal/Models/                          ComponentSlot (internal mounted-slot runtime object; implements both IGuiComponentSlot and IGuiRenderHandle, owns the child builder reference, arranged bounds, composed slot callbacks, and subtree invalidation/cascade lookup forwarding), GuiDialogRuntime (internal implementation of IGuiDialogRuntime used by dialogs for open/close/focus/cursor requests without depending on DialogRenderer input methods), InteractiveRegion (internal; bounds + token + six mouse callbacks + OnMouseWheel; HasClickHandlers distinguishes click-capable from wheel-only registrations; Translated(dx,dy) offsets bounds and carries all callbacks), ResizeRegion (internal; arranged bounds + slot token + IGuiResizable target recorded automatically for resizable slots), KeyboardRegion (internal readonly struct — slot identity token + OnKeyDown/Up/Press and OnFocusChanged callbacks; no bounds, keyboard and focus are not spatially routed), ScopedRebuildQueue (internal; dirty-frame queue of scoped reconcile tokens, drained before arrange/paint), FloatingLayerPlacement (internal readonly record struct — placement config consumed by FloatingLayerRenderer: Anchor (FloatingLayerAnchor), FixedLogicalSize, MaxLogicalWidth/Height, InputHost (GuiSurfaceRenderer?), InputRegionOffsetX/Y, AutoClearWhenNotRefreshed, RewalkOnDialogWalk)
Internal/Common/                          CascadingValueChain (internal; immutable single-linked list of providers), GuiResizeCursors (internal static; lazy on-demand registration of resize-edge cursor sprites — ships NS/EW/diagonal PNGs that vanilla omits; EnsureLoaded(clientApi) is idempotent; exposes cursor-code constants Horizontal/Vertical/DiagonalNwSe/DiagonalNeSw)
Internal/Enums/                           GuiMouseEventKind (internal), GuiKeyEventKind (internal; Down/Up/Press)
Internal/Delegates/                       FloatingLayerAnchor (internal; floating-layer position resolver: (physicalWidth, physicalHeight, scale, clientApi) → (posX, posY) physical-pixel top-left; all size/position params are double; consumed by FloatingLayerPlacement.Anchor)
Internal/Dialogs/                         VanillaGuiDialog             thin wrapper over VS GuiDialog; base for legacy vanilla-composer dialogs
Internal/Adapters/                        GuiElementAdapter            direct vanilla GuiDialog adapter registered for a framework GUI element; keeps DialogsOpened > 0, inherits vanilla draw-order/focus behavior, drives DialogRenderer.OnRenderFrame, and forwards mouse/keyboard/focus/Escape events to GuiInputRouter
Internal/Extensions/                      ClientApiExtensions          internal; reflection-based UnregisterDialog helper
```
